# Werewolf 5E

## Project Description

Werewolf 5E is a role-playing game adaptation inspired by classic mythology and horror themes. This project aims to provide a comprehensive set of rules, character sheets, and resources for players and game masters to create and enjoy immersive werewolf-themed gaming experiences.

## Setup Instructions

To set up the project locally, please follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/RNBWspite/Werewolf5e.git
   cd Werewolf5e
   ```

2. **Install Dependencies**
   Depending on the tools and libraries used in this project, you may need to install specific dependencies. Refer to the project's documentation for detailed instructions.

3. **Run the Application**
   Follow the instructions in the documentation to start the game or view the character sheets.
